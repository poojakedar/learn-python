# Day 1 Concepts: Variables and Data Types

Python stores information in variables.
A variable is like a labeled box where you keep a value.

## 1) What is a variable

You create a variable by writing:

name = "Pooja"

- name is the variable name
- "Pooja" is the value

## 2) Common data types

1. str (string): text
- Example: "hello"

2. int (integer): whole number
- Example: 10

3. float: decimal number
- Example: 5.4

4. bool: true/false
- Example: True, False

## 3) print function

Use print() to display output.

Example:
print("Hello")

You can print many values together:
print("Age:", 20)

## 4) type function

Use type() to check data type.

Example:
x = 5
print(type(x))

## 5) Reassigning variables

You can change value later.

age = 20
age = 21

Now age stores 21.

## 6) Variable naming rules

1. Use letters, numbers, underscore
2. Cannot start with a number
3. No spaces in name
4. Names are case-sensitive
- age and Age are different

Good names:
- user_name
- total_marks
- is_student

## 7) Common beginner mistakes

1. Forgetting quotes around text
- Wrong: city = Mumbai
- Correct: city = "Mumbai"

2. Mixing up type values
- "20" is string
- 20 is integer

3. Using unclear names
- Prefer score over x

## 8) Practice checklist

After Day 1 you should be able to:

1. Create variables of all 4 basic types
2. Print values
3. Check type with type()
4. Update variable value
